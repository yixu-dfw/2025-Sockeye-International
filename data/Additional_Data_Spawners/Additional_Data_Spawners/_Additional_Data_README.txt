This zip file includes a reorganized version of all the data shared for the 2025 Sockeye International in a single file: GENERATED_FullDataSet_LongForm.csv. This file includes estimates by age class. Variables are described in the file 3_VariableDescriptions_For_FullDataSet.txt. In addition, a spawner-recruit version is provided: GENERATED_SRDataSet_LongForm.csv

Additional summary files are also included (e.g., age class conversion table, main age class by age). 

Note the disclaimer: these are provided as a supplementary data set, in case it is useful to participants. In case of any discrepancies, use the official files instead!
